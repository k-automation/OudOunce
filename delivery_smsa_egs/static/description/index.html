Odoo - SMSA Express Shipping Connector

With Odoo - SMSA Express shipping connector solution of Egregious Solutions you can manage all Shipping operations with SMSA.
Automatically order information will be submit from Odoo to SMSA Express and get Shipping label, and Order Tracking number from SMSA Express to Odoo.

Odoo SMSA connector require below library to install

sudo pip install zeep

Features

  • Odoo eCommerce and Backend integration with SMSA Express.
  • Calculate shipping rate from sales order
  • Generate Shipping Request to SMSA Express from Odoo
  • Generate Tracking information from SMSA Express
  • Generate and auto attach shipping label to delivery order
